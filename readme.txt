== Description ==

### Perso Slider                       

Beautiful images and videos slider, responsive and retina, autoplay, touch devices, youtube, vimeo, keek, instagram support, compatible with all major browsers.

### The Features

* Fully Responsive.
* Retina Ready.
* Unlimited amount of images and videos to be added to the slider. 
* Unlimited Sliders.
* Autoplay Support (will be stopped when hover).
* Custom wrap width and wrap height.
* YouTube and Vimeo and Keek support.
* Instagram images support.
* Images Captions.
* Touch devices support.
* Compatible with any lightbox.
* Compatible with Google Chrome, FireFox, Opera, Safari, IE9, IE10, IE11.
* Easy to use and without options page, 2 shortcodes only.

== Installation ==

### Installation

1. Upload 'perso-slider' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Perso Slider" in wp-admin menu.
