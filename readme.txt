== Description ==

### Perso Slider

Responsive and Retina images and videos slider, simple but flexible with touch devices support and compatible with Google Chrome, FireFox, Opera, Safari, IE9, IE10, IE11.

### The Features

* Fully Responsive.
* Retina Ready.
* Unlimited amount of images and videos to be added to the slider.
* Unlimited Sliders.
* YouTube and Vimeo support.
* Images Captions.
* Touch devices support.
* Compatible with any lightbox.
* Compatible with Google Chrome, FireFox, Opera, Safari, IE9, IE10, IE11.
* Easy to use and without options page, 2 shortcodes only.

== Installation ==

* Upload 'perso-slider' folder to the '/wp-content/plugins/' directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Use perso slider shortcodes: [persoslider_w][persoslider_c img="" before_img="" cap="" before_cap="" video=""][/persoslider_w]
* This shortcode is slider wrap [persoslider_w] [/persoslider_w]
* This shortcode is your content [persoslider_c] put it inside slider wrap shortcode.
* [persoslider_c] Attributes:
1. img="here add your image link, don't add text" (this attributes is required).
2. before_img="here add link for your image, example add your post link or leave blank, don't add text" (this attributes is not required).
3. cap="your text caption, add text only, don't add link" (this attributes is not required).
4. before_cap="here add link for your caption, example add your post link or leave blank, don't add text" (this attributes is not required).
5. To add video, use this attribute only: video="here is your video link, youtube or vimeo only" without another attributes.
* Enjoy.
